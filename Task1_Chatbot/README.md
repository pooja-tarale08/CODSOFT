# Student AI ChatBot

## Description

A simple rule-based chatbot developed using Python. The chatbot responds to predefined user queries related to greetings, motivation, jokes, study tips, Python, DSA, date, and time.

## Features

* Greeting responses
* Motivational quotes
* Random programming jokes
* Python and DSA learning guidance
* Current date and time display
* Study tips
* Exit command support

## Technologies Used

* Python
* Random Module
* Datetime Module

## How to Run

1. Install Python on your system.
2. Open the project folder.
3. Run the following command:

```bash
python chatbot.py
```

## Sample Commands

* hello
* your name
* motivation
* joke
* python
* dsa
* time
* date
* study
* bye
